# Buyside Digest / Yet Another Value Podcast notes — Verdad Biotech

Source: https://www.buysidedigest.com/podcast/investing-in-biotech-with-verdad-capital/
Tier: context only

## Specialist definition
- Specialist = >50% of portfolio in biotech
- ~70 specialist funds in Verdad dataset; funds drop in/out automatically
- No performance filter on specialists; consensus voting machine
- ETFs (e.g. ARK Genomic) = one vote

## Consensus insight
- Names with zero specialists ~zero returns over time
- Multiple specialists outperform unique single-fund ideas
- Relative specialist density (specialists / all holders) matters for small caps

## Other factors discussed
- Spend-based value (Revenue - CFO) vs market cap; often stronger than specialist for rebalancing
- Insider: buys only; CFO/non-CEO; counts not dollars; multi-month horizon
- Shorts: diversified, SI/borrow, dampen vol
- Peer momentum via clinical trial similarity clusters
- PIPE/warrants and strategic pharma stakes = research pipeline

## Portfolio construction
- Frequent rebalancing on relative scores, not event prediction
- 13F lag acceptable due to illiquidity / long holds
