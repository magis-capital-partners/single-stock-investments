# Hohn Business Analysis Layer

**Genius:** Chris Hohn / TCI Fund Management  
**Purpose:** Operating-level business analysis with quantified thesis pillars and explicit return bridges — the analytical depth in TCI investor letters (2004–2018).

**Placement in decision stack:** Runs inside **Business & moat** (mechanics + evidence) and **Payoff & return** (valuation bridge). Does not replace Lawrence IRR math or Munger moat judgment.

**Sources:** `_system/reference/investment-wisdom/tci/` — 32 quarterly letters extracted from TCI Letters Portfolio.pdf. Curated patterns: `tci/Hohn-Analysis-Framework-extract.txt`.

## What Hohn adds that other layers miss

| Layer | Question | Hohn adds |
|-------|----------|-----------|
| Stahl | What archetype? | **How** the franchise earns: volume × price × mix, fee stack, regulatory pass-through |
| Munger | Will moat last? | **Operating proof:** trend data, peer gaps, management meetings, disintermediation watch |
| Lawrence | 10yr IRR at price? | **Medium-term bridge:** EPS path → exit multiple → annualized return; alternate methods (NAV, cap rate, FCF yield) |
| Pabrai | Dhando asymmetry? | **Bounded downside math:** stand-alone worth, catalyst date, ~10% downside vs 25–30%+ upside screen |

Hohn writes like an **operating analyst + activist owner**: recent quarter → structural drivers → management fieldwork → quantified return case → named primary risk.

## Hohn essentials (every deep dive — keep these simple)

Distilled from TCI letters. **Do these in prose + numbers;** do not add framework labels in the narrative.

| # | Rule | Example |
|---|------|---------|
| 1 | **Show the return math in words** | "8% FCF yield + 5–7% growth ≈ 14% total return" (Oracle); "$40 × 15× = $600 in 3 years → 27% p.a." (Charter) |
| 2 | **State upside / downside from today's price** | "Base payoff $18 from $6.70 = ~169% total upside; bear $9.5 ≈ 42% upside; book $8.55 caps fundamental floor" |
| 3 | **2–4 thesis pillars, each quantified** | Volume trough → recovery %; cost program €X → Y% EPS; legacy re-pricing over N years |
| 4 | **Label structural vs cyclical** in one short paragraph on the pillars | Not only in a table footnote |
| 5 | **One primary risk** | Single named risk in returns statement and first line of Risks; at most 3 secondary bullets |
| 6 | **Fieldwork or honest gap** | "None this period" + what meeting/data would change conviction |
| 7 | **Normalize one-offs** | Rebates base, unrealized marks, provisions — separate run-rate |

**Operating companies:** latest quarter vs prior quarter vs trend; **% changes** on volume, price, margin. Name one peer when comparing cost, yield, or multiple.

**Holding companies / optionality:** parent may have no operating P&L — then **look-through snapshot** (top 3–5 stakes) + **sum-of-parts or NAV table** in the report body (not only in [HUMAN REVIEW]). **Catalyst path:** 2–3 bullets with timing (Altaba-style: tender date, tax cliff, next liquidation step).

**Target:** ~15% p.a. medium-term on base case; screen **~25–30%+ upside / ~10% downside** from price where applicable (maps to Pabrai dhando, not a separate lecture).

## Entry criteria (Hohn screen — use at onboard / stance)

From TCI Q1 2004 and recurring letters:

| Criterion | Hohn bar | Marvin mapping |
|-----------|----------|----------------|
| Business quality | High barriers; monopoly/oligopoly/franchise | Stahl archetype + Munger moat |
| Return target | ~**15% p.a.** medium-term total return | Lawrence `implied_irr` base case |
| Upside / downside | **25–30%+ upside**, ~**10% downside** on longs | Pabrai dhando + Hohn valuation bridge |
| Valuation first | Invest on **valuation case + franchise**, catalyst optional | Five-question gate #4–5 |
| Absolute, not relative | Low beta; not dependent on market rerating alone | HK predictive attribute when dated payoff exists |

**Shorts (when relevant):** stand-alone worth well below price + identifiable catalyst (e.g. rights issue, capital hole). Document in Risks & inversion — not required for every Marvin name.

## Position analysis template (required in deep dives)

Use under **Business & moat → Business mechanics (Hohn)**. Cite `{TICKER}/path/to/file.pdf` with page/section.

### 1. Operating snapshot

| Field | Content |
|-------|---------|
| Latest period | Revenue, EBITDA/operating income, EPS — **YoY and vs prior quarter** |
| Volume / activity | Units, payment volume, passengers, listings — **trough → recovery** if cyclical |
| Pricing | List price, net yield, take rate, tariff — **% change YoY** |
| Mix / margin | Segment or product mix shift; incremental margin on volume (rails, exchanges) |
| One-off vs run-rate | Provisions, rebates base effects, integration costs — **normalize explicitly** |

*Example (TCI 2010 Q1, Visa): payment volume troughed Q2 2009; revenue lagged volume by a quarter; rebates base effect flagged for H2.*

### 2. Thesis pillars (2–4 structural drivers)

Name each pillar; attach **numbers and timeline**.

| Pillar type | Ask | Example |
|-------------|-----|---------|
| **Reversionary pricing** | Legacy contracts rolling off? | UP coal contracts; rail legacy re-pricing over 5 years |
| **Operating gap vs peers** | Cost/unit, margin, ROCE vs best peer? | UP cost/unit +2.5% vs CSX +6% |
| **Structural volume** | Secular growth independent of cycle? | Eurex +23% p.a. decade; debit mix shift |
| **Cost / synergy program** | €X cuts = Y% EPS if revenue flat? | Deutsche Börse €100m → ≥15% EPS growth 2008 |
| **Capital return** | Buyback, dividend, balance-sheet capacity? | Aena excess debt capacity → €11bn distributable |
| **Regulatory / political balance** | Itemized pros and cons | Wellpoint healthcare bill — net roughly balanced |
| **Hidden / zero-valued asset** | Subsidiary priced at zero in multiple? | Mining co utility; Waymo at zero in Alphabet |
| **Catalyst path** | Dated event unlocking value? | Fox auction; Altaba tender → liquidation |

### 3. Fieldwork & management

| Item | Document |
|------|----------|
| Investor day / site visit / mgmt meeting | What changed conviction? |
| Management claims vs independent check | Sell-side wrong how? |
| Governance / alignment | Incentive-caused bias (Munger cross-check) |

*Hohn norm: Omaha (UP), India bank trip, F-35 facility, Airbus CEO — record in deep dive when primary docs silent.*

### 4. Competitive & disruption watch

- **Structural vs cyclical** — label each headwind (2007 exchange volumes: structural growth + volatility tailwind).
- **Disintermediation** — mobile, online, vertical integration (Visa vs PayPal; Coke bottler buy).
- **Peer strategic moves** — competitor action forcing response (Pepsi bottlers → Coke CCE deal).

### 5. Valuation bridge (required table)

At least **two methods**; one must show **explicit return math**.

| Method | Inputs | Output |
|--------|--------|--------|
| **Earnings yield + growth** | Forward P/E or yield; 5–7% FCF growth → total return | Oracle: 8% FCF yield + 5–7% → ~14% |
| **EPS path × exit multiple** | EPS Y0 → Y3; exit P/E → price → **annualized IRR** | Charter: $40 FCF/sh 2021 × 15x → 27% 3yr |
| **FCF yield vs bonds** | FCF yield vs sovereign/corp bond | J-Power 15% FCF vs 1.5% JGB |
| **NAV / RAB / cap rate** | Regulated RAB + unregulated cap rate | Aena €34bn vs €23bn mkt cap |
| **Liquidation / holdco discount** | Pre-tax NAV, tax rate, blockage | Altaba 24% discount to NAV |
| **Normalized cycle earnings** | Peak/trough/mid on activity | Exchange on 20x vs double-digit growth |

```markdown
| Case | Method | Key inputs | Implied return | vs Hohn 15% bar |
|------|--------|------------|----------------|-----------------|
| Base | EPS path × exit multiple | … | …% p.a. | pass / fail |
| Bear | … | … | …% | … |
| Bull | … | … | …% | … |
```

### 6. Returns statement (one line + numbers)

Hohn always closes with an explicit expectation:

> *"We expect 15–20% long-term returns based on [pillars]. Biggest risk: [X]."*

Map base case to Lawrence `implied_irr` — they should reconcile or explain divergence in [HUMAN REVIEW].

### 7. Primary risk (named)

One **dominant** risk — not a laundry list (UP: double-dip recession; CSX: near-term economy vs long-term pricing).

## Archetype-specific Hohn prompts

Load with Tier 2 from `archetype_models.json` → `hohn_prompts`.

| Archetype | Extra Hohn questions |
|-----------|---------------------|
| **croupier / platform** | Volume vs share? Fee yield trajectory? Cyclical normalization on activity? Disintermediation channel? |
| **compounder** | Pricing vs volume split? Reinvestment rate vs FCF return? M&A EBIT math (Sun: $5bn, $1.5bn EBIT Y1)? If **≥2 reportable segments**, add segment economics table and point to `segment_cashflow_valuation.md` (Alphabet: Search / YouTube / Cloud / Waymo zero base — TCI Q2 2018). |
| **infrastructure** | Regulated vs merchant return; RAB/cap rate; traffic vs opex leverage (Aena: +9% traffic, +1% opex)? |
| **turnaround** | Provisioning peak? NIM path? Capital ratio floors and CoCo triggers? Catalyst date? |
| **holding_co** | **Required:** look-through table + SOTP/NAV vs price + catalyst path with dates; Altaba-style discount % and liquidation/tender steps |
| **serial_acquirer** | Synergy €/timeline; pro forma multiple on fully synergized earnings? |
